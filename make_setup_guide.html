✅ Checklist de Ativação

  • Criar conta no make.com (gratuito)
  • Criar novo cenário → adicionar módulo Webhook → copiar URL
  • Colar URL em CONFIG.MAKE_WEBHOOK_URL no index.html
  • Publicar index.html no GitHub Pages
  • Criar planilha "Alpha - Pedidos de Faturamento" no Google Sheets
  • Conectar Google Drive e Google Sheets ao Make.com
  • Criar conta Z-API → conectar instância WhatsApp pelo QR Code
  • Enviar pedido teste pelo formulário
  • Verificar: linha criada na planilha + WhatsApp recebido
  • Ativar o cenário (toggle ON no Make.com)
